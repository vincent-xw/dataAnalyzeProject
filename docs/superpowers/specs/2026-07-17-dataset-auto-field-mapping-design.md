# 数据集首工作表与自动字段映射设计

## 目标

减少数据集上传后的重复操作：Excel 工作簿默认检查第一个工作表；进入字段映射页后，依据模板已有的英文标准字段名和中文说明自动预选安全的一对一映射。

## 范围

仅修改前端上传与字段映射体验，不新增 Worker 接口、数据库字段或 LLM 调用。CSV 仍沿用现有编码、分隔符和检查流程。

## 工作表流程

上传 XLSX 后，若 Worker 返回工作表目录，前端将第一个工作表写入选择状态并立即以该名称调用检查接口。检查成功后直接进入字段映射页；用户仍可在上传页重新上传其他文件，映射页不提供隐式切换工作表。

## 自动映射规则

字段映射页初始化时，将来源表头、模板字段 `name` 与 `description` 分别规范化：去除首尾空格，英文转小写，并忽略空格、下划线和连字符差异。

来源表头仅在唯一命中一个模板字段时自动预选。若一个来源表头匹配多个模板字段，或多个来源表头命中同一个模板字段，则相关字段全部保持“忽略”。未匹配字段保持“忽略”。用户可以继续手动调整所有选择；既有必填字段和一对一映射校验继续生效。

## 错误与测试

首工作表检查失败时，沿用现有上传错误提示，不进入映射页。新增前端测试覆盖：多工作表默认检查首表、中文 `description` 匹配、英文 `name` 的规范化匹配，以及歧义和重复命中不自动映射。
