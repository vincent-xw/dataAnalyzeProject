# 图表空值与统计信息展示设计

## 目标

新建分析将空分组值统一显示为“空值”，并在所有图表中固定展示数据标签、tooltip 与 legend，使统计人数可直接阅读。

## 数据聚合

图表按维度和系列读取值时，将 `null`、`undefined`、空字符串、仅空白字符和 `-` 归一为“空值”。空值仍参与 `count` 或 `sum` 聚合，不会被过滤。因而图表准确呈现数据缺失规模，而不是静默丢弃记录。

## 图表展示

- 饼图：legend 使用维度分类；标签和 tooltip 显示 `名称：数值（百分比）`。
- 折线/柱状图：legend 使用 `series` 字段；tooltip 显示分类、系列与数值。没有 series 时隐藏 legend，避免无意义的空图例。
- 对 count 聚合，tooltip 和轴标题统一使用“人数”；sum 聚合使用“数值”。

## 系统提示词

新增默认提示词版本。它保留 count/series 协议，并要求模型把存在空值的维度按“空值”语义描述；图表的 tooltip、legend 与标签由渲染系统自动提供，不让模型生成协议外字段。

## 范围

只影响新创建分析的系统提示词和全部分析的图表显示。用户将删除旧分析；不对其冻结规则做数据迁移。
