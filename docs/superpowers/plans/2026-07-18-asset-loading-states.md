# 数据资产加载状态 Implementation Plan

**Goal:** 为资产列表和上传页提供可见、不可重复提交的加载状态。

### Task 1: 列表加载状态
- [ ] 写失败测试，验证请求 pending 时显示“正在加载数据资产…”。
- [ ] 增加 `loading` 状态，仅在完成后显示空状态。

### Task 2: 上传状态
- [ ] 写失败测试，验证 pending 上传禁用控件并显示阶段文案。
- [ ] 增加 `uploading` 与 `phase` 状态，在 finally 中恢复。

### Task 3: 验证
- [ ] 运行资产前端范围测试与类型检查。
